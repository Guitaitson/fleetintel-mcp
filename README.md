# FleetIntel MCP

MCP Server para consultas inteligentes em dados de frotas via Supabase + Redis.

## Status do Projeto
🚧 Em desenvolvimento - Fase 0: Planejamento

## Arquitetura
- **FastAPI**: MCP Server
- **Supabase**: Database principal
- **Redis**: Cache + Rate limiting
- **LangGraph**: Agente orquestrador
- **Evolution API**: Interface WhatsApp

## Branches
- `main`: Produção
- `dev`: Staging
- `feature/*`: Features em desenvolvimento

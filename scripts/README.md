# Scripts Administrativos

Utilitários para setup, migração e manutenção.

## Scripts Planejados

- `init_db.py`: Criar tabelas e índices no Supabase
- `seed_dev.py`: Popular DB com dados de teste
- `reset_cache.py`: Limpar cache Redis
- `check_health.py`: Validar todas integrações
- `migrate_data.py`: Migração de schemas (futuro)
